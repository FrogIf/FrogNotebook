正则表达式

A:字符

x 字符 x。举例：'a'表示字符a
\\ 反斜线字符。//在java中 \有特殊的含义，如果要打印该字符，必须两个\一起
\r 回车符 ('\u000D') //回车表示光标放在了一行的开头
\n 新行（换行）符 ('\u000A')  //回到下一行

B:字符类

[abc] a、b 或 c（简单类） 
[^abc] 任何字符，除了 a、b 或 c（否定） 
[a-zA-Z] a到 z 或 A到 Z，两头的字母包括在内（范围） 
[0-9] 0到9的字符都包括

C:预定义字符类

. 任何字符。我的就是.字符本身，怎么表示呢? \.
\d 数字：[0-9]
\w 单词字符：[a-z_A-Z_0-9]

D:边界匹配器

^ 行的开头
$ 行的结尾

E:Greedy 数量词

X?     X，一次或一次也没有
X*     X，零次或多次
X+     X，一次或多次
X{n}   X，恰好 n 次 
X{n,}  X，至少 n 次 
X{n,m} X，至少 n 次，但是不超过 m 次
